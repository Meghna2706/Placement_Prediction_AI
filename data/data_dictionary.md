# Data Dictionary

| Feature            | Description                        |
|------------------|--------------------------------------|
| cgpa             | Student GPA (5.0 - 10.0)             |
| num_internships  | Number of internships completed      |
| college_tier     | 1 = Top, 2 = Mid, 3 = Low            |
| job_domain       | Industry preference                  |
| is_metro_based   | 1 = Metro, 0 = Non-Metro             |
